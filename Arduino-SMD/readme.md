#Homemade Arduino SMD
##A clone of the Arduino-Pro-Mini

###Specifications

* Arduino Uno bootloader
* 3.3 volts @ 16MHz
* ATMega328 with all pins accessible
* LED on pin 5 (not pin 13)
* Onboard reset button
* FTDI programing headers
